## Brief 

All groups were given a week to create a fullstack MERN App using a custom build Backend API.

## Technologies
-   **Back-End**:
    
    -   Express
    -   MongoDB
    -   Node.js
    - Backend API  [here](https://peekback.herokuapp.com/api)

-   **Front-End**:
    
    -   CSS3 + SASS
    -   HTML5
    -   JavaScript (ES6)
    -   React.js
    -  Google Fonts
    -  Bulma
-   **Dependencies**:
    
    -   Axios
    -   bcrypt
    -   JSONWebToken
    -   mongoose-unique-validator
    -   npm
    -   react-router-dom

-   **Development Tools**:
    -   Git + GitHub
    -   Heroku
    -   VS Code
